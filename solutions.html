Solutions

soundDrop is a Secure Modular Based System which processes product data into Key Features, called ‘Books’. Each user has their own logon page in which their own details such as price, mark ups and discounts are securely assigned so that all information it tailored and accessible only to that user.

Key Features :

searchBook Collates all product data from all supplier catalogues and maps that data into common fields to enable users to search items from one golden source of data. This means that availability, distributor and price are readily available for all products and negates the need to search on spreadsheets. Includes ‘Add to orderBook’ & ‘Add Alerts’ (for stock changed items).

priceBook Enables the Retailer to securely assign all pricing information from each respective supplier, where that information retrieved and calculated for each individual item. These include Discounts, Mark-ups, Rounding information and VAT. Secure user controls ensures only staff with the correct rights have access to sensitive data.

orderBook Lists / displays all orders by Status, and Add/Select items, or Enables information recalled from searchBook to be added in one click, so that items can be easily ordered and submitted to Suppliers. The orderbook is designed so that no manipulation of data is required as the order information auto-populates into the format requested by Suppliers.

compareBook Compares data between catalogue information for each supplier over specified data ranges, so that users can readily identify new items, items out of stock, back in stock, deletions and price changes. This feature enables retailers to more easily manage items that they do not physically stock, and gain advantage of additional sales opportunities.

reportBook A dashboard tool that slices and dices all information, with metrics from ordering, products or pricing being readily available.
